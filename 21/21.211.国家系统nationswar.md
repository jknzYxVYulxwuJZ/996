### 国家宣战
`nationswar`

| 参数   | 类型      | 空   | 默认 | 注释                      |
| :----- | :-------- | :--- | :--- | :------------------------ |
| play   | `object`  | 否   |      | 玩家对象                  |
| nIdx   | `integer` | 否   |      | 国家ID                    |
| iValue | `integer` | 否   |      | 时间,单位小时(0=立即关闭) |
```lua
    nationswar(actor,5, 2000)
```
