### 人物显示一个放大的虚影

`showphantom`

| 参数    | 类型      | 空   | 默认 | 注释          |
| :------ | :-------- | :--- | :--- | :------------ |
| play    | `object`  | 否   |      | 玩家对象      |
| opacity | `integer` | 否   |      | 透明度(0~255) |
| time    | `integer` | 否   |      | 显示时间(秒)  |

```lua
showphantom(actor,100,3)
```
