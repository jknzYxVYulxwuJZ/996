### 发送聊天框消息

`sendmsg`

| 参数 | 类型      | 空   | 默认 | 注释                                                         |
| :--- | :-------- | :--- | :--- | :----------------------------------------------------------- |
| play | `object`  | 否   |      | 玩家对象                                                     |
| type | `integer` | 否   |      | 发送对象：<br />1-自己，2-全服<br />3-行会，4-当前地图<br />5-组队 |
| msg  | `string`  | 否   |      | Json消息内容                                                 |
