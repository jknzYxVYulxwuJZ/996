### 根据名称获取地图基础信息
`getmapinfo`

| 参数   | 类型      | 空   | 默认 | 注释                   |
| :----- | :-------- | :--- | :--- | :--------------------- |
| mapID  | `string`  | 否   |      | 地图ID                 |
| nIndex | `integer` | 否   |      | 0：地图宽<br />1：地图高 |
| result | `integer` | 否   |      | 返回地图基础信息       |
