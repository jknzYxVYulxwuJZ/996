### 修改攻击模式
`changeattackmode`

| 参数       | 类型      | 空   | 默认 | 注释                                                                                                                                     |
| :--------- | :-------- | :--- | :--- | :--------------------------------------------------------------------------------------------------------------------------------------- |
| play       | `object`  | 否   |      | 玩家对象                                                                                                                                 |
| attackmode | `integer` | 否   |      | 攻击模式：<br />0-全体攻击<br />1-和平攻击<br />2-夫妻攻击<br />3-师徒攻击<br />4-编组攻击<br />5-行会攻击<br />6-红名攻击<br />7-国家攻击<br />8-阵营攻击 |
