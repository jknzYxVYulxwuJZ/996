### 设置当前攻击目标
`settargetcert`

| 参数   | 类型     | 空   | 默认 | 注释                         |
| :----- | :------- | :--- | :--- | :--------------------------- |
| Hiter  | `object` | 否   |      | 攻击者<br />(玩家/英雄/怪物)   |
| Target | `object` | 否   |      | 被攻击者<br />(玩家/英雄/怪物) |
```
    将Hiter当前的攻击目标设定为Target
    注意：玩家实际攻击目标由客户端锁定，以修改当前玩家攻击目标为目的会失效。
```
