#### 受击前

`struckdamage` 玩家受击
`struckdamagebb` 宝宝受击
`herostruckdamage` 英雄受击
`herostruckdamagebb` 英雄宝宝受击

| 参数    | 类型      | 空   | 默认 | 注释                 |
| :------ | :-------- | :--- | :--- | :------------------- |
| self    | `object`  | 否   |      | 玩家对象             |
| Hiter   | `object`  | 否   |      | 攻击对象             |
| Target  | `object`  | 否   |      | 受击对象             |
| MagicId | `integer` | 否   |      | 技能ID               |
| Damage  | `integer` | 否   |      | 伤害                 |
| result  | `integer` | 否   |      | 返回值，修改后的伤害 |
