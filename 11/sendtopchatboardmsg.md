### 发送聊天框固顶信息
`sendtopchatboardmsg`

| 参数     | 类型      | 空   | 默认 | 注释                                                             |
| :------- | :-------- | :--- | :--- | :--------------------------------------------------------------- |
| player   | `object`  | 否   |      | 玩家对象                                                         |
| type     | `integer` | 否   |      | 发送对象<br />0-所有人<br />1-自己<br />2-行会<br />3-当前地图<br />4-组队 |
| FColor   | `integer` | 否   |      | 字体景色                                                         |
| BColor   | `integer` | 否   |      | 背景色                                                           |
| time     | `integer` | 否   |      | 显示时间，自动替换内容中的%d                                     |
| msg      | `string`  | 否   |      | 消息内容                                                         |
| showflag | `integer` | 否   |      | 是否显示人物名称<br />0-是<br />1-否                                 |
