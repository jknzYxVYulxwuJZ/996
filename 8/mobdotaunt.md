### 宝宝嘲讽

`mobdotaunt`

| 参数     | 类型      | 空   | 默认 | 注释                                                 |
| :------- | :-------- | :--- | :--- | :--------------------------------------------------- |
| play     | `object`  | 否   |      | 玩家对象                                             |
| idx      | `integer` | 否   |      | 第几个宝宝（第一个宝宝为0）                          |
| rang     | `integer` | 否   |      | 距离格子数                                           |
| levelMax | `integer` | 否   |      | 受嘲讽影响的怪物等级上限（不大于指定等级均会被吸引） |
```lua
    mobdotaunt(actor,0 ,20 ,1000)
```
