### 检测地图逻辑格
`gridattr`

| 参数   | 类型      | 空   | 默认 | 注释                                                                    |
| :----- | :-------- | :--- | :--- | :---------------------------------------------------------------------- |
| mapid  | `string`  | 否   |      | 地图Id                                                                  |
| x      | `integer` | 否   |      | x坐标                                                                   |
| y      | `integer` | 否   |      | y坐标                                                                   |
| type   | `integer` | 否   |      | 逻辑格类型:<br />1.能否到达;<br />2.安全区; <br />3.攻城区;                   |
| result | `boolean` | 否   |      | 地图逻辑格的实际属性是否与指定属性类型相同<br />true:相同<br />false:不相同 |
