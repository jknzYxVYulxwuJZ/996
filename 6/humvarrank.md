### 取自定义数字变量名位置

`humvarrank`

| 参数     | 类型     | 空   | 默认 | 注释                  |
| :------- | :------- | :--- | :--- | :-------------------- |
| actor    | `object` | 否   |      | 人物对象              |
| var      | `string` | 否   |      | 排序变量名            |
| playflag | `int`    | 否   |      | 0-所有玩家 1-在线玩家 |
| sortflag | `int`    | 否   |      | 0-升序，1-降序        |
| result   | `int`    |      |      | 人物的名次            |
