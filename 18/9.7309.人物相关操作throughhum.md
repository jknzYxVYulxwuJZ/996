### 设置玩家穿人穿怪
`throughhum`

| 参数    | 类型      | 空   | 默认 | 注释                                         |
| :------ | :-------- | :--- | :--- | :------------------------------------------- |
| play    | `object`  | 否   |      | 玩家对象                                     |
| type    | `integer` | 否   |      | 模式：0-恢复默认；1-穿人；2-穿怪；3-穿人穿怪 |
| time    | `integer` | 否   |      | 时间(秒)                                     |
| objtype | `integer` | 否   |      | 对象 ：0-玩家；1-宝宝                        |
