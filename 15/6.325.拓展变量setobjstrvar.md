#### 设置对象str变量

&gt; **临时变量,不会存储数据库,重载NPC后变量丢失**

`setobjstrvar`

| 参数  | 类型      | 空   | 默认 | 注释          |
| :---- | :-------- | :--- | :--- | :------------ |
| obj   | `object`  | 否   |      | 怪物/NPC对象  |
| idx   | `integer` | 否   |      | 变量位置(1-5)|
| value | `string`  | 否   |      | 变量值        |
