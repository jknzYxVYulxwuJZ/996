### 播放音乐声音
`playsound`

| 参数  | 类型      | 空   | 默认 | 注释                                                                              |
| :---- | :-------- | :--- | :--- | :-------------------------------------------------------------------------------- |
| play  | `object`  | 否   |      | 玩家对象                                                                          |
| index | `integer` | 否   |      | 播放文件的索引<br />对应声音配置表id(cfg_sound.xls)                                 |
| times | `integer` | 否   |      | 循环播放次数                                                                      |
| flag  | `integer` | 否   |      | 播放模式:<br />0.播放给自己<br />1.播放给全服<br />2.播放给同一地图<br />4.播放给同屏人物 |
