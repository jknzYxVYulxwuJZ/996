### 添加足迹特效

`setmoveeff `

| 参数     | 类型      | 空   | 默认 | 注释                                     |
| :------- | :-------- | :--- | :--- | :--------------------------------------- |
| play     | `object`  | 否   |      | 玩家对象                                 |
| effectID | `integer` | 否   |      | 特效id                                   |
| modle    | `integer` | 否   |      | 播放模式<br />0/nil=两步播放<br />1=单步播放 |
```lua
    setmoveeff(actor,17,1)
```
