### 播放屏幕特效
`screffects`

| 参数     | 类型      | 空   | 默认 | 注释                           |
| :------- | :-------- | :--- | :--- | :----------------------------- |
| player   | `object`  | 否   |      | 玩家对象                       |
| id       | `integer` | 否   |      | 创建的特效编号                 |
| effectid | `integer` | 否   |      | 特效ID                         |
| X        | `integer` | 否   |      | 在屏幕上的X坐标                |
| Y        | `integer` | 否   |      | 在屏幕上的Y坐标                |
| speed    | `integer` | 否   |      | 播放速度                       |
| times    | `integer` | 否   |      | 播放次数，0-持续播放           |
| type     | `integer` | 否   |      | 播放模式<br />0-自己<br />1-所有人 |
