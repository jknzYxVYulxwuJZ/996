### 设置聊天前缀
`setchatprefix`

| 参数   | 类型      | 空   | 默认 | 注释                       |
| :----- | :-------- | :--- | :--- | :------------------------- |
| object | `object`  | 否   |      | 玩家对象                   |
| Prefix | `string`  | 否   |      | 前缀信息，空则清除聊天前缀 |
| color  | `integer` | 否   |      | 背景色                     |
