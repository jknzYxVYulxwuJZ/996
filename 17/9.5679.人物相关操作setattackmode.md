### 强制修改攻击模式
`setattackmode`

| 参数       | 类型      | 空   | 默认 | 注释                            |
| :--------- | :-------- | :--- | :--- | :------------------------------ |
| play       | `object`  | 否   |      | 玩家对象                        |
| attackmode | `integer` | 否   |      | 攻击模式<br />-1-提前结束强制状态 |
| time       | `integer` | 否   |      | 强制切换时间时间                |
