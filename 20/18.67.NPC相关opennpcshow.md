### 打开指定NPC面板
`opennpcshow`

| 参数     | 类型     | 空   | 默认 | 注释                       |
| :------- | :------- | :--- | ---- | -------------------------- |
| play     | `object` | 否   |      | 玩家对象                   |
| NPCIndex | `int`    | 否   |      | NPC索引（NPC配置表中的ID） |
| nRange   | `int`    | 否   |      | 范围值，在此范围内允许打开 |
