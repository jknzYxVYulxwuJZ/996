### 设置宠物蛋等级
`setpetegglevel`

|参数|类型|空|默认|注释|
|:----    |:-------    |:--- |----|------      |
|play     |`object`    |否   |    |   玩家对象 |
|itemmakeid     |`integer`    |否   |    |   物品MakeIndex |
|level     |`integer`    |否   |    |   等级，-1表示不修改值|
|zlevel     |`integer`    |否   |    |   转生等级，-1表示不修改值|
|exp     |`integer`    |否   |    |   经验值，-1表示不修改值|
