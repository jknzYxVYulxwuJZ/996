### 给按钮删除红点

`reddel`

| 参数   | 类型      | 空   | 默认 | 注释                  |
| :----- | :-------- | :--- | :--- | :-------------------- |
| play   | `object`  | 否   |      | 玩家对象              |
| win_id | `integer` | 否   |      | 窗口ID                |
| btn_id | `integer` | 否   |      | 按钮ID/任务栏填任务ID |

```lua
reddel(actor, 104, 7)
```
