### 设置人物伤害吸收
`setsuckdamage`

| 参数    | 类型      | 空   | 默认 | 注释                                         |
| :------ | :-------- | :--- | :--- | :------------------------------------------- |
| play    | `object`  | 否   |      | 玩家对象                                     |
| operate | `string`  | 否   |      | 操作符号<br />'+'-增加<br />'-'-减少<br />'='-等于 |
| sum     | `integer` | 否   |      | 总吸收量                                     |
| rate    | `integer` | 否   |      | 吸收比率，千分比<br />1=0.1%，100=10%          |
| success | `integer` | 否   |      | 吸收成功率                                   |
