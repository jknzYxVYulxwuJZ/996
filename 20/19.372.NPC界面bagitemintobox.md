### 把包裹中的物品放入自定义OK框中

`bagitemintobox`

| 参数  | 类型               | 空   | 默认 | 注释                |
| :---- | :----------------- | :--- | :--- | :------------------ |
| actor | `object`           | 否   |      | 玩家对象            |
| item  | `string`/`integer` | 否   |      | 物品名称<br>物品唯一id |
| idx   | `integer`          | 否   |      | OK框编号(0~99)      |
