| 参数  | 类型     | 空   | 默认                | 注释     |
| :---- | :------- | :--- | :------------------ | :------- |
| actor | `object` | 否   |                     | 玩家对象 |
| item  | `object` | 否   | 引擎64_23.08.30新增 | 物品对象 |
| name  | `string` | 否   | 引擎64_23.08.30新增 | 物品名字 |
