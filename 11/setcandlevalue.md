### 设置人物照亮范围（光照）

`setcandlevalue`

&gt; **&lt;font color="#808080" style="font-size: 13px;"&gt;引擎64_24.05.23新增接口&lt;/font&gt;**

| 参数  | 类型      | 空   | 默认 | 注释                                |
| :---- | :-------- | :--- | :--- | :---------------------------------- |
| actor | `object`  | 否   |      | 玩家对象                            |
| value | `integer` | 否   |      | 人物照亮范围值<br />-1=读装备的光照值 |

```
    注:黑夜模式需在`M2-假人设置-光照系统-是否免蜡`中开启
```
