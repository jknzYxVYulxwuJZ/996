### 人物变色
`setbodycolor`

| 参数  | 类型      | 空   | 默认 | 注释                                                                     |
| :---- | :-------- | :--- | :--- | :----------------------------------------------------------------------- |
| play  | `object`  | 否   |      | 玩家对象                                                                 |
| color | `integer` | 否   |      | 颜色(0~255); 255时清除颜色;     -1则为转生设置的颜色在人物身体上进行变色 |
| time  | `integer` | 否   |      | 改变时长(秒)                                                             |
