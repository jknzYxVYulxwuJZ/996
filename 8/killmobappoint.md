### 在指定位置优先打指定打怪
`killmobappoint`

|参数|类型|空|默认|注释|
|:----    |:-------    |:--- |----|------      |
|player |`object`      |否   |    | 玩家对象 |
|map |`string`      |否   |    | 地图 |
|X |`integer`      |否   |    | X坐标 |
|Y |`integer`      |否   |    | Y坐标 |
|MonName |`string`      |否   |    | 优先攻击的怪物名称 |
```
MonName支持多个怪物名称，怪物名称中间用 | 分隔
```
