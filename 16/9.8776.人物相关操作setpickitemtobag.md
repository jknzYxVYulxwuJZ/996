### 拾取道具飞入背包按钮动作

`setpickitemtobag`

| 参数   | 类型      | 空   | 默认 | 注释     |
| :----- | :-------- | :--- | :--- | :------- |
| play   | `object`  | 否   |      | 玩家对象 |
| win_id | `integer` | 否   |      | 窗口ID   |
| btn_id | `integer` | 否   |      | 按钮ID   |

```lua
setpickitemtobag(actor,104,7)
```
