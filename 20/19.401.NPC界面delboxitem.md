### 删除自定义OK框中的物品
`delboxitem`

| 参数  | 类型      | 空   | 默认 | 注释                                                  |
| :---- | :-------- | :--- | :--- | :---------------------------------------------------- |
| actor | `object`  | 否   |      | 玩家对象                                              |
| idx   | `integer` | 否   |      | OK框编号(0~99)                                        |
| num   | `integer` | 否   |      | 删除数量;参数只有是叠加物品时才会有效，为空则全部删除 |
