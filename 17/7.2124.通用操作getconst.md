### 获取常量
`getconst`

| 参数    | 类型     | 空   | 默认 | 注释                                                 |
| :------ | :------- | :--- | :--- | :--------------------------------------------------- |
| player  | `object` | 否   |      | 玩家对象                                             |
| varname | `string` | 否   |      | 常量名称，支持带尖括号和不带尖括号<br />&lt;$Name&gt;或$Name |
| result  | `string` | 否   |      | 常量值                                               |
