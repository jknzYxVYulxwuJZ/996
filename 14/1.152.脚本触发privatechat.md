### 收到私聊信息触发

`privatechat`

| 参数   | 类型      | 空   | 默认 | 注释         |
| :----- | :-------- | :--- | :--- | :----------- |
| self   | `object`  | 否   |      | 玩家对象     |
| sMsg   | `string`  | 否   |      | 说话内容     |
| chat   | `integer` | 否   |      | 聊天频道     |
| result | `bool`    | 否   |      | 是否接收私信 |
------------
