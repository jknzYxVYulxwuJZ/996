### 初始化行会自定义变量

`iniguildvar`

| 参数    | 类型     | 空   | 默认 | 注释                     |
| :------ | :------- | :--- | :--- | :----------------------- |
| guild   | `object` | 否   |      | 行会对象                 |
| varType | `string` | 否   |      | 变量类型(integer/string) |
| varName | `string` | 否   |      | 变量名                   |
