### 删除宠物攻击表现

`delpetskill`

| 参数    | 类型      | 空   | 默认 | 注释                                      |
| :------ | :-------- | :--- | :--- | :---------------------------------------- |
| play    | `object`  | 否   |      | 玩家对象                                  |
| idx     | `integer` | 否   |      | 宠物序号或'X'表示当前宠物                 |
| skillid | `integer` | 否   |      | 增加的攻击表现ID，为cfg_monattack表中的ID |
