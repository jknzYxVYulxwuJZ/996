| msgid  | `integer` | 否   |      | 消息ID   |
| param1 | `integer` | 是   |      | 参数1    |
| param2 | `integer` | 是   |      | 参数2    |
| param3 | `integer` | 是   |      | 参数3    |
| sMsg   | `string`  | 是   |      | 消息体   |
