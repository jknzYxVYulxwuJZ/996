### 设置人物永久属性
`setusebonuspoint`

| 参数   | 类型      | 空   | 默认 | 注释       |
| :----- | :-------- | :--- | :--- | :--------- |
| play   | `object`  | 否   |      | 玩家对象   |
| nIndex | `integer` | 否   |      | 索引       |
| nvalue | `integer` | 否   |      | 对应属性值 |

nIndex 取值
1：攻击下限（0~65535）
2：攻击上限（0~65535）
3：魔法下限（0~65535）
4：魔法上限（0~65535）
5：道术下限（0~65535）
6：道术上限（0~65535）
7：防御下限（0~65535）
8：防御上限（0~65535）
9：魔防下限（0~65535）
10：魔防上限（0~65535）
11：生命值（支持21亿）
12：魔法值（支持21亿）
13：准确（支持21亿）
14：躲避（支持21亿）
15：防御下限（支持21亿）
16：防御上限（支持21亿）
17：魔防下限（支持21亿）
18：魔防上限（支持21亿）
