### 打开NPC大窗口
`openmerchantbigdlg`

| 参数   | 类型  | 空   | 默认 | 注释                        |
| :----- | :---- | :--- | ---- | --------------------------- |
| path   | `str` | 否   |      | 路径                        |
| pos    | `int` | 否   |      | 显示位置                    |
| x      | `int` | 否   |      | X坐标                       |
| y      | `int` | 否   |      | Y坐标                       |
| height | `int` | 否   |      | 高度                        |
| width  | `int` | 否   |      | 宽度                        |
| bool   | `int` | 否   |      | 是否显示关闭按钮            |
| closeX | `int` | 否   |      | 关闭按钮X坐标               |
| closeY | `int` | 否   |      | 关闭按钮Y坐标               |
| isMove | `int` | 否   |      | 是否可以移动(0不移动 1移动) |
