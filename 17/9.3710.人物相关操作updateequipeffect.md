### 给人物装备面板加特效
`updateequipeffect`

| 参数     | 类型      | 空   | 默认 | 注释                    |
| :------- | :-------- | :--- | :--- | :---------------------- |
| play     | `object`  | 否   |      | 玩家对象                |
| effectid | `integer` | 否   |      | 特效ID， 0-删除特效     |
| position | `integer` | 否   |      | 显示位置：0-前面 1-后面 |
