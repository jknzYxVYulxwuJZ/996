### 加入行会前触发

`guildaddmember`

&gt; 需要申请入会的情况,会触发两次:申请时触发一次,会长同意申请后再触发一次

| 参数   | 类型     | 空   | 默认 | 注释                            |
| :----- | :------- | :--- | :--- | :------------------------------ |
| play   | `object` | 否   |      | 玩家对象                        |
| guild  | `object` | 否   |      | 行会对象                        |
| name   | `stirng` | 否   |      | 行会名称                        |
| result | `bool`   | 是   |      | true=允许加入<br />false=阻止加入 |
