### 设置地图杀怪经验倍数
`mapkillmonexprate`

| 参数  | 类型      | 空   | 默认 | 注释                                                                                                   |
| :---- | :-------- | :--- | :--- | :----------------------------------------------------------------------------------------------------- |
| play  | `object`  | 否   |      | 玩家对象                                                                                               |
| MapId | `string`  | 否   |      | 地图id( * 号表示所有地图)                                                                              |
| much  | `integer` | 否   |      | 倍率 为杀怪经验倍数，倍数除以100为真正的倍率(200 为 2 倍经验，150 为1.5倍,0表示关闭地图的杀怪经验倍数) |
