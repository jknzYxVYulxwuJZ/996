### 判断地图坐标是否为空
`isemptyinmap`

| 参数   | 类型      | 空   | 默认 | 注释                 |
| :----- | :-------- | :--- | :--- | :------------------- |
| mapID  | `string`  | 否   |      | 地图ID               |
| nX     | `integer` | 否   |      | 地图x坐标            |
| nY     | `integer` | 否   |      | 地图y坐标            |
| result | `bool`    | 否   |      | 返回地图坐标是否为空 |
