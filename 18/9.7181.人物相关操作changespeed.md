### 改变玩家速度
`changespeed`

| 参数  | 类型      | 空   | 默认 | 注释                                                                      |
| :---- | :-------- | :--- | :--- | :------------------------------------------------------------------------ |
| play  | `object`  | 否   |      | 玩家对象                                                                  |
| type  | `integer` | 否   |      | 速度类型：<br />1-移动速度<br />2-攻击速度<br />3-施法速度                      |
| level | `integer` | 否   |      | 速度等级 -10~10<br />0-原始速度，<br />-1时间间隔减少10%<br />+1时间间隔增加10% |
