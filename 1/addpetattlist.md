### 增加宠物属性

`addpetattlist`

| 参数     | 类型               | 空   | 默认 | 注释           |
| :------- | :----------------- | :--- | :--- | :------------- |
| play     | `object`           | 否   |      | 玩家对象       |
| idx      | `integer`          | 否   |      | 宠物序号       |
| attrName | `integer`/`string` | 否   |      | 自定义属性组名 |
| opt      | `string`           | 否   |      | 操作符 + - =   |
| attr     | `string`           | 否   |      | 属性字符串     |
