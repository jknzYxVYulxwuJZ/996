### 设置人物经验值
`changeexp`

| 参数   | 类型      | 空   | 默认 | 注释               |
| :----- | :-------- | :--- | :--- | :----------------- |
| play   | `object`  | 否   |      | 玩家对象           |
| opt    | `char`    | 否   |      | 操作符 + - =       |
| count  | `integer` | 否   |      | 数量               |
| addexp | `bool`    | 否   |      | 是否增加聚灵珠经验 |
