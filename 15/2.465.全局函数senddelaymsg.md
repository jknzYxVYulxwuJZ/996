### 显示倒计时信息提示
`senddelaymsg`

| 参数      | 类型      | 空   | 默认 | 注释                                 |
| :-------- | :-------- | :--- | :--- | :----------------------------------- |
| player    | `object`  | 否   |      | 玩家对象                             |
| msg       | `string`  | 否   |      | 消息内容                             |
| time      | `integer` | 否   |      | 时间，秒                             |
| FColor    | `integer` | 否   |      | 字体景色                             |
| mapdelete | `integer` | 否   |      | 换地图是否删除<br />0-不删除<br />1-删除 |
| tag       | `string`  | 否   |      | 跳转的函数字段                       |
| X         | `integer` | 否   |      | X坐标                                |
