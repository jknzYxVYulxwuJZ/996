### 修改武器、衣服特效
`changedresseffect`

| 参数    | 类型      | 空   | 默认 | 注释                                            |
| :------ | :-------- | :--- | :--- | :---------------------------------------------- |
| play    | `object`  | 否   |      | 玩家对象                                        |
| where   | `integer` | 否   |      | 位置 0，1                                       |
| EffId   | `integer` | 否   |      | 特效ID                                          |
| selfSee | `integer` | 否   |      | 是否只有自己看见(1=所有人都可见 0=仅仅自己可见) |
