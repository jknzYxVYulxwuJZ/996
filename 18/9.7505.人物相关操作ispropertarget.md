### 判断对象是否可被攻击
`ispropertarget`

| 参数   | 类型      | 空   | 默认 | 注释                                          |
| :----- | :-------- | :--- | :--- | :-------------------------------------------- |
| Hiter  | `object`  | 否   |      | 攻击对象<br />(玩家/英雄/怪物)                  |
| Target | `object`  | 否   |      | 被攻击对象<br />(玩家/英雄/怪物)                |
| Result | `boolean` | 否   |      | 返回值<br />true:可以被攻击<br />false:不可被攻击 |
