### 用脚本命令释放技能

`releasemagic`

|参数|类型|空|默认|注释|
|:----    |:-------    |:--- |---|------      |
|play     |`object`    |否   |    |   玩家对象 |
|skillid     | `integer`    |否   |    |   技能ID |
|type     | `integer`    |否   |    |   类型：<br />1-普通技能<br />2-强化技能 |
|level     | `integer`    |否   |    |  技能等级 |
|target     | `integer`    |否   |    | 技能对象：<br />1-攻击目标，<br />2-自身|
|flag     | `integer`    |否   |    |   是否显示施法动作：<br />0-不显示，1-显示 |
