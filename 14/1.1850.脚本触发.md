#### 攻击后

`attack` 玩家物理攻击
`magicattack` 玩家魔法攻击
`attackpet` 宝宝物理攻击
`magicattackpet` 宝宝魔法攻击
`heroattack` 英雄物理攻击
`heromagicattack` 英雄魔法攻击
`heroattackpet` 英雄宝宝物理攻击
`heromagicattackpet` 英雄宝宝魔法攻击

| 参数    | 类型      | 空   | 默认 | 注释     |
| :------ | :-------- | :--- | :--- | :------- |
| self    | `object`  | 否   |      | 玩家对象 |
| Target  | `object`  | 否   |      | 受击对象 |
| Hiter   | `object`  | 否   |      | 攻击对象 |
| MagicId | `integer` | 否   |      | 技能ID   |
