### 发送屏幕滚动信息
`sendmovemsg`

| 参数   | 类型      | 空   | 默认 | 注释                                                                   |
| :----- | :-------- | :--- | :--- | :--------------------------------------------------------------------- |
| player | `object`  | 否   |      | 玩家对象                                                               |
| type   | `integer` | 否   |      | 模式，发送对象<br />0-自己<br />1-所有人<br />2-行会<br />3-当前地图<br />4-组队 |
