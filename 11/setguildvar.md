### 设置自定义变量

`setguildvar`

| 参数    | 类型               | 空   | 默认 | 注释                                 |
| :------ | :----------------- | :--- | :--- | :----------------------------------- |
| guild   | `object`           | 否   |      | 行会对象                             |
| varName | `string`           | 否   |      | 变量名                               |
| value   | `string`/`integer` | 否   |      | 变量值                               |
| isSave  | `integer`          | 是   | 0    | 是否保存数据库<br>0=不存储<br>1=存储 |
