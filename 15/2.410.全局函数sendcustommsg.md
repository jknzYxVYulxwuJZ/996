### 屏幕任意坐标发送公告信息
`sendcustommsg`

| 参数   | 类型      | 空   | 默认 | 注释                                                           |
| :----- | :-------- | :--- | :--- | :------------------------------------------------------------- |
| player | `object`  | 否   |      | 玩家对象                                                       |
| type   | `integer` | 否   |      | 消息类型<br />0-全服<br />1-自己<br />2-组队<br />3-行会<br />4-当前地图 |
| msg    | `string`  | 否   |      | 消息内容                                                       |
| FColor | `integer` | 否   |      | 前景色                                                         |
| BColor | `integer` | 否   |      | 背景色                                                         |
| X      | `integer` | 否   |      | X坐标                                                          |
| Y      | `integer` | 否   |      | Y坐标                                                          |
