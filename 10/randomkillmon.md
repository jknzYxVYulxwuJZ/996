### 随机杀死地图中的怪物
`randomkillmon`

| 参数        | 类型      | 空   | 默认 | 注释                           |
| :---------- | :-------- | :--- | :--- | :----------------------------- |
| mapid       | `string`  | 否   |      | 地图Id                         |
| monstername | `string`  | 否   |      | 怪物名字                       |
| num         | `integer` | 否   |      | 数量(1-255)                    |
| obj         | `integer` | 否   |      | 掉落物品(0,1)  0=掉落 1=不掉落 |
