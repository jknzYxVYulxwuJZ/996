### 过滤全服提示信息
`filterglobalmsg`

| 参数   | 类型      | 空   | 默认 | 注释                           |
| :----- | :-------- | :--- | :--- | :----------------------------- |
| player | `object`  | 否   |      | 玩家对象                       |
| flag   | `integer` | 否   |      | 是否过滤<br />0-不过滤<br />1-过滤 |

```
开启过滤全服提示信息，不再接受如SENDMSG、GuildNoticeMsg等等脚本命令发送的全服提示信息。
```
