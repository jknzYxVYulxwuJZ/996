### 检测加入国家
`checknational`

| 参数   | 类型      | 空   | 默认 | 注释                              |
| :----- | :-------- | :--- | :--- | :-------------------------------- |
| play   | `object`  | 否   |      | 玩家对象                          |
| nIdx   | `integer` | 否   |      | 国家编号 0~100  0代表没有加入国家 |
| result | `bool`    | 是   |      | 返回值，布尔值                    |
