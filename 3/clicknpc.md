### NPC点击触发

`clicknpc`

| 参数   | 类型      | 空   | 默认 | 注释                                 |
| :----- | :-------- | :--- | :--- | :----------------------------------- |
| actor  | `object`  | 否   |      | 玩家对象                             |
| npcID  | `integer` | 否   |      | NPC索引（NPC配置表中的ID）           |
| npcName  | `string` | 否   |      | NPC名称                           |
| result | `bool`    | 否   |      | false允许继续执行Txt脚本，true不允许 |
------------
