### 查看别人面板信息
`viewplayer`

| 参数   | 类型      | 空   | 默认 | 注释                                  |
| :----- | :-------- | :--- | :--- | :------------------------------------ |
| player | `object`  | 否   |      | 玩家对象                              |
| userid | `string`  | 否   |      | 其他玩家的UserID                      |
| winID  | `integer` | 否   |      | 面板ID：101-装备，106-称号，1011-时装 |
