### 宠物脱装备

`pettakeoff`

> 此接口不扣减物品，仅扣减宠物身上对应装备属性。

| 参数 | 类型      | 空   | 默认 | 注释                                          |
| :--- | :-------- | :--- | :--- | :-------------------------------------------- |
| play | `object`  | 否   |      | 玩家对象                                      |
| idx  | `integer` | 否   |      | 宠物序号                                      |
| item | `string`  | 否   |      | 装备名称，多个装备用#分隔，-1表示脱下全部装备 |
