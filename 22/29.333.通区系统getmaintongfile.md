### 拉取文件(带主区命令)

`getmaintongfile`

| 参数     | 类型     | 空   | 默认 | 注释                                               |
| :------- | :------- | :--- | :--- | :------------------------------------------------- |
| serverID | `string` | 否   |      | 传入主区ID,在主区执行该命令无效                    |
| filePath | `string` | 否   |      | 本地文件路径`'..\\QuestDiary\\bbb.txt'`            |
| path     | `string` | 否   |      | 远程服务器路径<br />例 `'..\\QuestDiary\\996m2.txt'` |
