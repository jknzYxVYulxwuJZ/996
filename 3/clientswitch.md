### 前端勾选面板控制命令

`clientswitch`

| 参数 | 类型      | 空   | 默认 | 注释                                                                                         |
| :--- | :-------- | :--- | :--- | :------------------------------------------------------------------------------------------- |
| play | `object`  | 否   |      | 玩家对象                                                                                     |
| type | `integer` | 否   |      | 0=允许组队<br />1=允许添加好友<br />2允许交易<br />3=允许挑战<br />4允许查看<br />5=允许添加为行会成员 |
| time | `integer` | 否   |      | 1=允许(勾选)<br />0=不允许(不勾选)(秒)                                                         |

```lua
for i = 0, 5 do
    clientswitch(actor,i,1)
end
```
