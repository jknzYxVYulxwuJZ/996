### 判断对象是否为玩家
`isplayer`

&gt; **特殊:系统对象为0号玩家,所以也会返回true**

| 参数   | 类型     | 空   | 默认 | 注释                          |
| :----- | :------- | :--- | :--- | :---------------------------- |
| object | `object` | 否   |      | 判断对象                      |
| result | `bool`   | 否   |      | true=是玩家<br />false=不是玩家 |
