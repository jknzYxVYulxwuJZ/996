### 聊天触发前置接口

`triggerchat`

| 参数    | 类型      | 空   | 默认 | 注释                                                                                                                                                                                  |
| :------ | :-------- | :--- | :--- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| self    | `object`  | 否   |      | 玩家对象                                                                                                                                                                              |
| sMsg    | `string`  | 否   |      | 说话内容                                                                                                                                                                              |

| result  | `bool`    | 否   |      | 是否终止说话                                                                                                                                                                          |

function triggerchat(self, sMsg, chat,msgType)
