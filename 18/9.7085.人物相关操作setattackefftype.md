### 设定人物攻击飘血飘字类型
`setattackefftype`

| 参数 | 类型      | 空   | 默认 | 注释                                                                                                                                                              |
| :--- | :-------- | :--- | :--- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| play | `object`  | 否   |      | 玩家对象                                                                                                                                                          |
| type | `integer` | 否   |      | 显示类型<br /> 1- 伤害，<br />2- 暴击伤害，<br />4- 加HP，<br />5- 格挡，<br />8- 扣减HP和MP，<br />9- 伤害，<br />10-扣减MP，<br />11- 致命一击<br />对应cfg_damage_number表里的ID |

```
设定人物飘血飘字类型，传入参数为攻击者，当前攻击飘血时有效，一次设定后，即返回默认值
```
