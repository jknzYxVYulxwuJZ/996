### 检查国家人物总数
`checknationhumcount`

| 参数   | 类型      | 空   | 默认 | 注释           |
| :----- | :-------- | :--- | :--- | :------------- |
| play   | `object`  | 否   |      | 玩家对象       |
| sFlag  | `string`  | 否   |      | 比较符（=&lt;&gt;）  |
| iValue | `integer` | 否   |      | 人数           |
| result | `bool`    | 是   |      | 返回值，布尔值 |
```lua
    checknationhumcount(actor,'&gt;',100)
```
