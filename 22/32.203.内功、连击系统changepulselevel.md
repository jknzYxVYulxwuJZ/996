### 修改经络的修炼等级格式

`changepulselevel`

| 参数   | 类型      | 空   | 默认 | 注释                                                    |
| :----- | :-------- | :--- | :--- | :------------------------------------------------------ |
| player | `object`  | 否   |      | 玩家对象                                                |
| pulse  | `integer` | 否   |      | 经络<br />0=冲脉<br />1=阴跷<br />2=阴维<br />3=任脉3<br />4=奇经 |
| opt    | `string`  | 否   |      | 控制符(=,+,-)                                           |
| level  | `integer` | 否   |      | 等级                                                    |
