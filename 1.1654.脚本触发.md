#### 暴击触发

`crittrigger` 支持物理攻击、魔法攻击 暴击触发

| 参数    | 类型      | 空   | 默认 | 注释                 |
| :------ | :-------- | :--- | :--- | :------------------- |
| self    | `object`  | 否   |      | 玩家对象             |
| attack  | `object`  | 否   |      | 受击对象             |
| damage  | `integer` | 否   |      | 暴击后伤害值         |
| MagicId | `integer` | 否   |      | 技能ID               |
| result  | `integer` | 否   |      | 返回值，修改后的伤害 |
