### 增加动态地图连接
`addmapgate`

| 参数    | 类型      | 空   | 默认 | 注释                       |
| :------ | :-------- | :--- | :--- | :------------------------- |
| name    | `string`  | 否   |      | 连接名称                   |
| Mapfrom | `string`  | 否   |      | 地图ID                     |
| X1      | `integer` | 否   |      | X(小于0时随机坐标)         |
| Y1      | `integer` | 否   |      | Y(小于0时随机坐标)         |
| range   | `integer` | 否   |      | 范围                       |
| Mapto   | `string`  | 否   |      | 到达地图号                 |
| X2      | `integer` | 否   |      | 到达地图X(小于0时随机坐标) |
| Y2      | `integer` | 否   |      | 到达地图Y(小于0时随机坐标) |
| time    | `integer` | 否   |      | 有效时间秒                 |
