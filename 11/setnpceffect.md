### 设置NPC特效
`setnpceffect`

| 参数     | 类型     | 空   | 默认 | 注释                               |
| :------- | :------- | :--- | ---- | ---------------------------------- |
| player   | `object` | 否   |      | 玩家对象                           |
| NPCIndex | `int`    | 否   |      | NPC索引（NPC配置表中的ID）         |
| Effect   | `int`    | 否   |      | 特效ID<br>5055-感叹号<br>5056-问号 |
| X        | `int`    | 否   |      | X坐标                              |
| Y        | `int`    | 否   |      | Y坐标                              |

```
当X、Y<0时，默认取NPC的坐标
```
