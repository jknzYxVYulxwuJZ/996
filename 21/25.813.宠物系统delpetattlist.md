### 清除宠物属性

`delpetattlist`

| 参数     | 类型               | 空   | 默认 | 注释                                       |
| :------- | :----------------- | :--- | :--- | :----------------------------------------- |
| play     | `object`           | 否   |      | 玩家对象                                   |
| idx      | `integer`          | 否   |      | 宠物序号                                   |
| attrName | `integer`/`string` | 否   |      | 清空对应属性组的属性;<br>nil清除所有属性组 |
