#### 获取物品int变量[拓展];

`getitemintparam`

| 参数    | 类型      | 空   | 默认 | 注释                           |
| :------ | :-------- | :--- | :--- | :----------------------------- |
| actor   | `object`  | 否   |      | 玩家对象                       |
| where   | `integer` | 否   |      | 装备位置<br />(-2则传入物品对象) |
| idx     | `integer` | 否   |      | 变量位置(1-50)                 |
| itemobj | `object`  | 是   |      | 物品对象(参数2传入-2时有效)    |
| result  | `integer` | 否   |      | 变量值                         |
