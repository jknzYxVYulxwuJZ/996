### 怪物掉落物品触发

`mondropitemex` 怪物掉落统一触发

| 参数     | 类型      | 空   | 默认 | 注释                               |
| :------- | :-------- | :--- | :--- | :--------------------------------- |
| self     | `object`  | 否   |      | 玩家对象                           |
| DropItem | `object`  | 否   |      | 掉落物品                           |
| mon      | `object`  | 否   |      | 掉落怪物                           |
| nX       | `integer` | 否   |      | 掉落坐标x                          |
| nY       | `integer` | 否   |      | 掉落坐标y                          |
| result   | `bool`    | 否   |      | 是否掉落,true=允许掉落，false=不允许 |
