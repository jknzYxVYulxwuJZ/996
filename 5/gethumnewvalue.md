### 获取人物临时属性
`gethumnewvalue`

| 参数   | 类型      | 空   | 默认 | 注释                          |
| :----- | :-------- | :--- | :--- | :---------------------------- |
| play   | `object`  | 否   |      | 玩家对象                      |
| nWhere | `integer` | 否   |      | 位置 对应cfg_att_score 属性ID |
| result | `integer` | 否   |      | 对应属性值                    |
