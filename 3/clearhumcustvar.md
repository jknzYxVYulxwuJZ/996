### 清理个人自定义变量

`clearhumcustvar`

| 参数  | 类型     | 空   | 默认 | 注释                                        |
| :---- | :------- | :--- | :--- | :------------------------------------------ |
| actor | `object` | 否   |      | 要清理的人物对象<br>传入 * 表示清理所有玩家 |
| var   | `string` | 否   |      | 变量名, <br> * -所有变量                    |
```lua
    -- 多个变量用|分隔
    clearhumcustvar("*","*")
    clearhumcustvar(actor,"S变量1|S变量2")
```
