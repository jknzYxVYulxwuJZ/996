### 上马

`ridehorse`

| 参数        | 类型      | 空   | 默认 | 注释                               |
| :---------- | :-------- | :--- | :--- | :--------------------------------- |
| play        | `object`  | 否   |      | 玩家对象                           |
| HorseAppr   | `integer` | 否   |      | 坐骑外观                           |
| HorseEff    | `integer` | 否   |      | 坐骑特效外观                       |
| HorseFature | `integer` | 否   |      | 人物骑马外观                       |
| Type        | `integer` | 否   |      | 坐骑类型  0=单人   1=双人   2=连体 |
